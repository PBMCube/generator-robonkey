# Getting Started

Install yo

```sh
$ npm install -g yo
```

Install Robonkey

```sh
$ npm install generator-robonkey -g
```

Then you can start building your app with Robonkey:

```sh
$ mkdir yourapp
$ cd yourapp
$ yo robonkey
$ cd gulp
$ gulp
```

Make a custom Modernizr file:

```sh
$ gulp modernizr
```

Make a custom icon font:

```sh
$ gulp iconfont
```
