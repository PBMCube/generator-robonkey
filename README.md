#Robonkey Generator

> A [Yeoman](http://yeoman.io) front-end application generator with Jade, Sass &amp; Gulp

## Features

* Gulp
* Jade
* SCSS
* BrowserSync
* PostCSS
* Autoprefixr

## Options

* Javascript Libraries
	* jQuery
	* Waypoints
	* Signals
	* D3
	* TweenMax
	* Enquire
	* Google Analytics Script in body
* Jade
* Modernizr
* Custom Icon Font Generator	 
* PostCSS
	* CSSNano
	* MQPacker
	* MQKeyframes
	* SelectorNot
	* SelectorMatcher
* Prefix selectors `.myselector` -> `.prefix-myselector`
* Scope selectors `.myselector` -> `#scope .myselector`
* Base styles
	* Reset
	* Normalize
	* Sanitize
* SCSS Libraries
	* Susy
	* Breakpoint
* Boilerplate extra's
	* .htaccess
	* browserconfig.xml
	* crossdomain.xml
	* robots.txt and humans.txt
